Electron 主进程

基于 electron-vite 与 DI 容器，IPC 路由洋葱模型中间件，统一管理 Python 子进程与窗口生命周期

查看主进程文档

Vue 渲染端

Vue 3 + Pinia + Vue Router + Tailwind 4，配合 shadcn 风格组件库，按页面组织业务，可独立浏览器调试

查看渲染端文档

Python 后端

FastAPI 路由 + Presenter 模式，HTTP 同步业务调用与 WebSocket 反向通知一体化，支持守护进程与空闲软退出

查看后端文档

三通道通信

渲染层经 HTTP /api/bridge 调用 Python，Python 经 WebSocket /ws/electron 回调主进程，所有通道带超时与错误模型

守护进程模式

默认端口 47821 自动探测，元数据写入 LOCALAPPDATA，二次启动复用驻留进程，启动时间从 5 秒降到 1 秒

多平台打包

electron-builder 集成 PyInstaller，一条命令产出 Windows NSIS、macOS dmg、Linux AppImage/snap/deb
